# 📞 Grandstream GRP2614 – Configure Call Forwarding via Web GUI

# ✅ Option 1: Configure via Web GUI (Recommended)

## 📋 Prerequisites

Before starting, ensure you have:

* Access to the same network as the phone
* A web browser installed
* The phone administrator credentials

---

# 🛠️ Step-by-Step Configuration for GRP2614

## 1️⃣ Get the IP Address of the Phone

On the phone:

1. Press the **“OK”** button
   *(Center button of the navigation circle)*

2. The phone screen will display the IP address.

### Example

```text
192.168.1.50
```

---

## 2️⃣ Open a Web Browser

Open any web browser and enter the phone’s IP address into the address bar.

### Example

```text
http://192.168.1.50
```

---

## 3️⃣ Login to the Web Interface

Use the following credentials to sign in:

| Field        | Value                                                        |
| ------------ | ------------------------------------------------------------ |
| **Username** | `admin`                                                      |
| **Password** | Printed on the back label of the phone *(random by default)* |

> 🔒 **Note:** The default administrator password is device-specific and located on the phone label.

---

## 4️⃣ Navigate to Call Settings

From the web interface menu:

```text
Account → Call Settings
```

---

## 5️⃣ Configure Call Forwarding

Scroll down to the **“Call Forwarding”** section and apply the following settings:

| Setting                   | Configuration                          |
| ------------------------- | -------------------------------------- |
| **Forward No Answer**     | `ON`                                   |
| **Target**                | 123 [Enter User 2’s extension number]  *(Eg: Reception)* |
| **No Answer Timeout (s)** | `20` *(approximately 4 rings)*         |
| **Busy Forward**          | `OFF`                                  |
| **Unconditional Forward** | `OFF`                                  |

> 📌 **Tip:** A timeout value of **20 seconds** generally equals about **4 rings** before forwarding occurs.

---

## 6️⃣ Save the Configuration

Click:

```text
Save and Apply
```

The phone may briefly reload or re-register after applying the settings.

---

# ✅ Configuration Summary

| Feature               | Status     |
| --------------------- | ---------- |
| Forward No Answer     | Enabled    |
| Reception Extension   | Configured |
| Timeout               | 20 Seconds |
| Busy Forward          | Disabled   |
| Unconditional Forward | Disabled   |
