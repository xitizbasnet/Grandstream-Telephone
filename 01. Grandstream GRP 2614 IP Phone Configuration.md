## 1. Purpose
This document provides a standardized procedure for configuring the **Grandstream GRP 2614 IP telephone** within the organization’s VoIP infrastructure.  

The purpose of this guide is to assist **IT support engineers** and **system administrators** in properly configuring:
- SIP accounts  
- Codec settings  

Configuration is performed through the device’s **web management interface**.

> ✅ Following this procedure ensures:
> - Consistent configuration  
> - Proper SIP registration  
> - Reliable voice communication within the enterprise network

## 3. Device Information

The Grandstream GRP 2614 is a **professional enterprise-grade IP telephone** designed for **SIP-based VoIP communication**.

## 4. Telephone Setup
To initiate configuration, access the phone’s **web administration interface** using its IP address.

### 📋 Procedure:
- Ensure the phone is properly connected to the network.  
- Identify the IP address assigned to the device from the phone display.  
- Open a supported web browser from a workstation connected to the same network.  
- Enter the device IP address in the browser address bar.  

➡️ The **Grandstream login interface** will appear.

### 🔹 Step 1 – Login with IP  
Example: `192.168.1.120`

## 10. Codec Configuration
### 🔹 Step 7 – Go to Codec Setting

Codec configuration determines how **voice audio is encoded and transmitted** between endpoints.

### 📋 Procedure:
- Go to the **Codec Setting**  
- Add codec setting  

### 🎧 Allowed Codecs:
- **PCMU**  
- **PCMA**  

✔️ These codecs are widely supported and ensure compatibility with the SIP server infrastructure.

---

## 11. Apply Configuration
### 🔹 Step 8 – Apply Settings

After completing the codec configuration:

- Click **Save and Apply**

🔄 The device will:
- Apply the updated configuration  
- Refresh the SIP registration process (if required)
