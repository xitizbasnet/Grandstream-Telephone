# Purpose

# 🔧🔧 Steps to Configure Scheduled Reboot

---

## 1. Access the Web Interface

* Open a web browser.
* Navigate to the IP Phone login page:


---

## 2. Authenticate

### Password

```text id="s3ggn7"

```

* Click on the Login button.

---

## 3. Navigate to Scheduled Tasks

* From the left-side menu, go to Maintenance.
* Click on Scheduled Tasks.

---

## 4. Create a New Scheduled Task

* Click the + Add button to create a new schedule.
* Configure the necessary settings for the reboot (time, frequency, etc.) as per your requirements.

---

## 5. Apply Settings

* Once configured, click OK to save and apply the scheduled task.

---

## 6. Confirm Task Activation

* The newly created schedule should now appear in the dashboard list of scheduled tasks.
* Review and verify the task to ensure its applied correctly.

---

## 7. Final Verification

* Ensure the task appears correctly with the desired settings.
* Optionally, monitor the device after the scheduled time to verify successful reboot.

---

## 8. End of Procedure

Ensure that access to the web interface and administrative credentials are protected as per organizational IT security policy.
